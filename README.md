# 🎓 Uni Creative Marketplace

A responsive web-based marketplace for UK university students to connect with photographers and videographers.

## ✨ Features

- 🎓 Student-only access (.ac.uk email validation)
- 📸 Photographer/Videographer profiles
- 📅 Booking system
- 💳 Secure payments
- 💼 Commission-based platform earnings
- 🔐 Authentication & role-based access

---

## 🧱 Tech Stack

### Backend
- Laravel (API)
- MySQL
- JWT Authentication

### Frontend
- React (Vite)
- Tailwind CSS

### Payments
- Stripe / Paystack

---

## 🔐 User Roles

- Student (Client)
- Creative (Photographer/Videographer)
- Admin

---

## 📌 Core Functionalities

### Students
- Register with `.ac.uk` email
- Browse creatives
- Book services
- Make payments

### Creatives
- Create profile
- Set pricing
- Accept/reject bookings

### Platform
- Take commission on each transaction
- Manage users and bookings
